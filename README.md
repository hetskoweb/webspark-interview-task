# Webspark Interview Task

This project is a technical task aimed at demonstrating front-end development skills, including working with dynamic lists, date filtering, and view switching.

## Live demo

Experience the live website: [Websparktest](https://hetskoweb.github.io/webspark-interview-task/)

## 🚀 Features

- **Dynamic View Switching**: Toggle between "Grid" and "List" views for the posts list.
- **Date Filtering**: Filter posts by selecting a date range using the Flatpickr calendar library.
- **Responsive Design**: The interface is fully responsive and optimized for different screen sizes.

## 🛠 Tech Stack

- **HTML5** & **SCSS**
- **JavaScript** (ES6+)
- **Vite** (Build Tool)
- **Flatpickr**

## 📂 Project Structure

- `index.html`: Main entry point.
- `main.js`: Main logic for filtering, view switching, and calendar initialization.
- `styles/`: SCSS files for styling.
- `dist/`: Compiled production-ready files.

## 💡 Usage

1. Use the **View Switcher** buttons to change the layout.
2. Select dates in the **"From"** and **"To"** calendars to filter the displayed posts.
3. The calendar for "To" is initialized with today's date by default.

## 🧩 How to Run

1. **Clone the repository:**

```bash
git clone https://github.com/hetskoweb/webspark-interview-task.git
cd webspark-interview-task
```

2. **Install dependencies:**

```bash
npm install
# or
yarn install
```

3. **Run the project locally:**

```bash
npm run dev
# or
yarn dev
```

4. **Open the landing page in your browser:**

```bash
http://localhost:5173
```
